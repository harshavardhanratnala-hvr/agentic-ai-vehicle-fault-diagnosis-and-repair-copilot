# Agentic AI Vehicle Fault Diagnosis & Repair Copilot

Capstone project — neue fische Data Science & AI Bootcamp.

An agentic AI system that takes a vehicle fault trigger (a diagnostic trouble code or an anomalous sensor reading), classifies it, retrieves relevant NHTSA repair/safety records via RAG, and generates a structured, cited repair recommendation.

See `docs/Capstone_Project_Plan.md` for the full plan (source of truth — update this file directly when scope changes).

## Structure

- `docs/` — project plan, pitch materials, write-ups
- `notebooks/` — EDA and experimentation notebooks
- `src/` — Python source (classifier, RAG pipeline, agent, API)
- `slides/` — presentation decks
- `media/` — demo videos, screenshots
- `data/` — local datasets (gitignored — see `.gitignore`)
